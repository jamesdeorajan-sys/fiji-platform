## How this list grows

Every quarter, audit:
1. New hotel openings on Viti Levu
2. Closed properties (remove from list)
3. Renamed properties (e.g. "Hideaway" → "Hideaway Resort & Spa")
4. Lat/lng accuracy (test against Google Maps drive times)

When adding a new hotel:
1. Get lat/lng from Google Maps (right-click → "What's here")
2. Add to `app.js` destination dropdown AND pickup dropdown
3. Add to this MD file
4. If it's a major property, consider adding to the routes table
5. Test a sample booking from Nadi Airport → new hotel
6. Verify drive time matches Google Maps within 10%

When removing a hotel:
1. Comment out the option (don't delete) in `app.js` so existing bookings can still resolve
2. Mark as "[CLOSED]" here, don't delete
3. Add a redirect at `_redirects` if it had a deep link
